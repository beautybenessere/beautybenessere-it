# DESIGN SYSTEM — BEAUTYBENESSERE.IT

## Direzione generale
BeautyBenessere.it deve avere un design premium, elegante, caldo e leggibile.

La parte pubblica deve comunicare:
- bellezza;
- benessere;
- fiducia;
- professionalità;
- semplicità;
- territorialità.

## Front-end pubblico
Stile consigliato:
- sfondi avorio / bianco caldo;
- card bianche;
- accenti oro soft;
- verde salvia come supporto;
- testo antracite;
- bordi chiari;
- ombre leggere;
- foto grandi e curate;
- icone sottili;
- layout arioso;
- pulsanti leggibili.

## Palette indicativa
- Avorio caldo: #FAF7F2
- Bianco card: #FFFFFF
- Antracite testo: #232323
- Testo secondario: #6F6A63
- Oro soft: #C9A35A
- Verde salvia: #7FA99B
- Bordo chiaro: #E8DFD2
- Sfondo salvia chiaro: #EEF5F1

## Font consigliati
Possibili combinazioni:
- Titoli: Fraunces / Playfair Display / Cormorant
- Testi e UI: Inter / Manrope / Lato

Scelta consigliata iniziale:
- Titoli: Fraunces oppure Playfair Display
- UI/Testi: Inter oppure Manrope

## Leggibilità
Regola obbligatoria:
- mai testi sotto 14px salvo micro-dettagli;
- testo base almeno 16px;
- descrizioni importanti 16–18px;
- bottoni e input 16–18px;
- titoli card ben visibili;
- badge leggibili.

## Homepage pubblica
Elementi principali:
- header con logo Beauty Benessere;
- navigazione: Categorie, Città, Per gestori, Rivendica attività, Accedi;
- hero con titolo grande;
- barra ricerca: Cosa cerchi? Dove? Cerca;
- categorie principali;
- sezione mappa;
- città in evidenza;
- box per gestori;
- footer.

## Pagina risultati
Layout desktop:
- top search bar;
- filtri;
- lista attività a sinistra;
- mappa a destra.

Card attività:
- foto;
- nome attività;
- categoria;
- città;
- badge Verificato / In evidenza / Prenotazione online;
- descrizione breve;
- tag servizi;
- pulsanti Apri scheda / Chiama / Indicazioni.

## Scheda attività
Struttura consigliata:
- cover grande;
- logo;
- nome attività;
- categoria e città;
- badge;
- pulsanti Chiama, WhatsApp, Indicazioni, Prenota, Sito web;
- descrizione;
- servizi;
- galleria foto;
- orari;
- contatti;
- mappa;
- box “Sei il proprietario? Rivendica la scheda”.
